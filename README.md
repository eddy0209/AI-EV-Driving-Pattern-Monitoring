# AI-Based EV Driving Pattern Monitoring

This project simulates an AI-based electric vehicle driving pattern monitoring and range estimation system.

## Features
- Driving pattern classification
- Energy consumption estimation
- Remaining range prediction
- Streamlit-based virtual dashboard

## Technologies Used
- Python
- Streamlit
- Machine Learning
- ESP32 (planned hardware)

## Driving Modes
- Eco
- Normal
- Aggressive
